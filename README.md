# `gold`

**This is a toy project.**

`gold` is an in-memory key-value database that communicates using the `RESP` and `HTTP` protocols.

## Installation (stable)

```sh
cargo install gold-db
```

## Installation (git)

```sh
git clone https://github.com/minsk-dev/gold
cd gold
cargo install --path .
```

## Running

```sh
gold-db
```
